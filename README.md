# Free Hosting Options for Privacy Policy & Terms of Service

You don't need to pay for hosting! Here are **FREE** options:

## Option 1: GitHub Pages (Recommended - 5 minutes)

**Pros**: Free, reliable, easy, professional URL  
**Cons**: Requires GitHub account (free)

### Steps:
1. Create GitHub account: https://github.com
2. Create new repository: `habitflow-website`
3. Upload `privacy-policy.html` and `terms-of-service.html`
4. Go to Settings → Pages
5. Select branch: `main`
6. Your URLs will be:
   - `https://yourusername.github.io/habitflow-website/privacy-policy.html`
   - `https://yourusername.github.io/habitflow-website/terms-of-service.html`

## Option 2: Netlify Drop (Easiest - 2 minutes)

**Pros**: Drag & drop, instant, free  
**Cons**: URL includes "netlify.app"

### Steps:
1. Go to https://app.netlify.com/drop
2. Drag the `website` folder
3. Get instant URLs:
   - `https://random-name-123.netlify.app/privacy-policy.html`
   - `https://random-name-123.netlify.app/terms-of-service.html`

## Option 3: Vercel (Fast - 3 minutes)

**Pros**: Fast, free, good performance  
**Cons**: Requires account (free)

### Steps:
1. Go to https://vercel.com
2. Sign up (free)
3. Import project → Upload folder
4. Deploy
5. Get URLs automatically

## Option 4: Google Sites (Simple - 5 minutes)

**Pros**: Very easy, Google account  
**Cons**: Less professional URL

### Steps:
1. Go to https://sites.google.com
2. Create new site
3. Copy/paste content from HTML files
4. Publish
5. Get URL like: `sites.google.com/view/habitflow`

## Option 5: Use Your Own Domain (If you have one)

If you own a domain (e.g., `habitflow.app`):
1. Use any free hosting above
2. Point your domain to the hosting
3. Use: `https://habitflow.app/privacy` and `https://habitflow.app/terms`

## Update App Store Connect

Once you have URLs, add them to:
- App Store Connect → Your App → App Information → Privacy Policy URL
- Settings in your app (already configured to open these URLs)
